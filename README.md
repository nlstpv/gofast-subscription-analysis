# Scooter Sharing Service Analysis

## Project Overview

This project analyzes user behavior in a scooter sharing service to understand how subscription plans affect customer behavior and company revenue.

The goal is to provide data-driven recommendations for increasing subscription adoption and improving business performance.

---

## Business Task

The product team wants to answer several questions:

- Do subscribers spend more time riding?
- Do subscribers generate higher monthly revenue?
- Does subscription usage match the company's operational assumptions?
- Should the company invest in promoting subscriptions?

---

## Dataset

The analysis uses three datasets:

- Users
- Rides
- Subscriptions

---

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
- Jupyter Notebook

---

## Project Workflow

1. Data loading
2. Data cleaning
3. Exploratory Data Analysis
4. Revenue calculation
5. Statistical hypothesis testing
6. Business recommendations

---

## Key Findings

- Subscription users spend significantly more time riding.
- Monthly revenue from subscribers is higher.
- Subscription users are more valuable for the business.
- Marketing efforts should focus on increasing subscription conversion.
